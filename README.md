
# BMS

Introducing BMS MERN stack-powered bank management system app!

Experience real-time banking with facilities for both User and Admin. 


## Features

**User Features**
- Customer Login
- Customer Dashboard
- Fund Transfer
- Transaction History
- Apply Loan
- Pay Loan
- Loan History

**Admin Features**

- Admin Login
- Admin Dashboard
- Create User
- Delete User
- Approve Loan
- Pay Loan
- Deposit
- Withdraw
- Create Admin
